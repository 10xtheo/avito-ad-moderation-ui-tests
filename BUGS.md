# Баг-репорты для платформы модерации объявлений

ID: BUG-ADS-FILTER-PRICE-RANGE-1 (сбито на 5000)
Title: Фильтрация при поиске объявлений: сдвиг в плюс 5000 рублей при указании фильтра Диапазон цен (От)
Precondition:
1. В системе есть объявления с ценой до 5000р
STR: 
1. Перейти на главную страницу
2. В селекторе "Сортировать по" указать "Цене", в селекторе "Порядок" - "По возрастанию"
3. В блоке фильтров указать в Диапазоне цен (от) значение 0
Expected: Отображается список объявлений, содержащий в начале объявления с ценой до 5000 руб включительно
Actual: Отображается список объявлений, содержащий в начале объявления с ценой не менее 5000 руб
